CLASSE

Classe, no contexto de orientação a objetos, pode ser interpretada como molde.

OBJETO

Objeto, olhando por essa perspectiva, é algo gerado pelo molde. 


		class Palhaco:
		    """Molde para o palhaço"""
		    def __init__(self, nome, cor):
		        self.nome = nome
		        self.cor = cor
